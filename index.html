Natural Disasters and Earth Surface Temperature Analysis

The goal of this project is to better understand natural disaster patterns. Our team of five bootcamp students performed an efficient ETL process by extracting raw data from our sources, cleaning the data with Pandas and Jupyter Notebook, and storing our transformed data in an AWS database and a S3 bucket, which was then connected to PostgreSQL and back into Jupyter Notebook. Our analysis continued with three different machine learning models, which included one supervised logistic regression technique and two Facebook Prophet techniques. These models attempt to predict natural disaster and earth temperature patterns. Our analysis concluded with an interactive Tableau dashboard to display our visualizations.

Cryptocurrencies with Unsupervised Machine Learning

This analysis dives into cryptocurrency data and unsupervised machine learning techniques using Python, Pandas, and Jupyter Notebook. I use the technique of K Means Clustering from Scikit-learn to discover how cryptocurrencies could be grouped to create a classification system. The machine learning results on the cryptocurrency data are displayed using Plotly packages.

Vehicle Analysis with R

This statistical analysis uses R to analyze data sets on prototype vehicles. I focus on multiple linear regression analyses to help identify which variables in the prototype dataset can predict MPG. I then collect summary statistics on the pounds per square inch (PSI) of the suspension coils from the manufacturing lots, and run t-tests to determine if the manufacturing lots are statistically different from the mean population.

Biodiversity with JavaScript

This interactive dashboard was developed using HTML, JS, and D3 to read in data on volunteer bacterial samples in JSON format. The dashboard contains a drop down menu which enables the user to select a specific volunteer ID. Once this ID is selected from the drop down, demographic info and the dashboard visualizations will change to display data specific to that ID.

Hawaii Climate Analysis with Python and SQLite

Temperature trends in Oahu, Hawaii are analyzed to determine if the location is a profitable area for business investors to open a successful combination surf and ice cream shop. I specifically looked at the months of June and December, to determine is these months will help the surf and ice cream shop be sustainable year round. This analysis uses Python, Pandas, and Jupyter Notebook to explore the data, which is loaded through SQLite and SQLAlchemy. A preliminary analysis was also conducted using Python, Pandas, and Jupyter Notebook, and can be displayed on a local server using Flask.
